Detailed Description

This abstract class declares method signatures for any class that would like to be the channel back to the "client", either on a remote client system, the server system itself or whatever.

The main idea is to get around the problem of having Firmata related code way down in the guts of the device drivers. Using this class, any sort of client-facing asynchronouse message processing can be supported without affecting the device drivers.
